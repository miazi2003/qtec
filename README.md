# QTEC Job App


## Requirements
- Node.js 18+
- npm 9+

## Setup
1. Install dependencies:
```bash
npm install
```
2. Create `.env.local`:
```env
VITE_API_BASE_URL={BACKEND_URL}/api
```

## Run locally
```bash
npm run dev
```
Then open: `http://localhost:5173`

## Build for production
```bash
npm run build
npm run preview
```
